## Description
In the Categories configuration UI, add a column that displays how many activities are using each category. This gives
users a quick overview of category usage before editing or deleting.

## Acceptance criteria
- [ ] Categories table has a new "Activities" column
- [ ] Each row shows the count of activities referencing that category
- [ ] Count is read-only (display only)
- [ ] Count reflects current data (fetched alongside categories or via a separate query)
- [ ] Column renders correctly when count is 0
